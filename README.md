# Isaac Sim 4.5 Examples

This repository contains custom examples and scripts for Isaac Sim 4.5.

## Setup

1. Navigate to your Isaac Sim 4.5.0 installation directory:
```bash
cd /path/to/isaac-sim-4.5.0
```

2. Clone this repository:
```bash
git clone https://github.com/v-xchen-v/isaac_sim_4.5_xc.git
```

3. Run the scripts directly:
```bash
cd isaac_sim_4.5_xc
python get_pose_of_a_single_link.py
```

The scripts will use the Isaac Sim Python environment and dependencies from the parent installation directory.

## Requirements

- Isaac Sim 4.5.0 installed
- Scripts must be run from within the Isaac Sim installation directory structure
